## Development

`git clone git@github.com:will-stone/browserosaurus.git`

`cd browserosaurus`

`yarn install`

`yarn start`

### Browser logos

https://github.com/alrra/browser-logos

### Make icon.icns

To build an icns file from `src/images/icon/icon.png`, simply run `yarn run icns`.
